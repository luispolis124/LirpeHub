# LirpeHub 📺
**"Broadcast Yourself"** – A video platform that revives the nostalgic aesthetic of the 2005 Web 2.0 era.
## 🚀 About the Project
**LirpeHub** was created to bring back the simplicity and charm of video platforms from the early 2000s. Developed to be an accessible social network focused on original content and community, it blends a retro visual style with cutting-edge technology for streaming and interaction.
## ✨ Key Features
 * **Hybrid Player:** Intelligent playback that switches between YouTube and custom streaming servers (HLS/MP4) using *Video.js*.
 * **Rating System:** Real-time star-based rating system for videos.
 * **Interactivity:** Dynamic comments and content reporting system.
 * **Scalable Backend:** Powered by *Supabase* (Authentication, Database, and Realtime).
 * **Nostalgic Design:** User interface inspired by the social media "boom" of 2005.
## 🛠 Technologies Used
 * **Frontend:** HTML5, CSS (Web 2.0 style), Vanilla JavaScript.
 * **Player:** *Video.js* (for complete compatibility with video formats).
 * **Backend/BaaS:** *Supabase* (PostgreSQL).

## ⚙️ Development Setup
To run the project locally:
 1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/lirpehub.git
   
   ```
 2. Create a js/config.js file in the js/ folder with your Supabase credentials:
   ```javascript
   const CONFIG = {
       SUPABASE_URL: "YOUR_SUPABASE_URL",
       SUPABASE_ANON_KEY: "YOUR_SUPABASE_ANON_KEY"
   };
   
   ```
 3. Open index.html in any web browser.
